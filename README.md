# Asteroids Clone

A simple Asteroids-style arcade game built with `pygame` as part of a Boot.dev project.

## How to Run

```bash
pip install -r requirements.txt
python main.py

Controls
WASD: Move
Space: Shoot
Esc / window close button: Quit
Features
Player movement and shooting
Asteroid spawning and splitting
Score increases when asteroids are destroyed

Adjust commands/controls to match your game. If you want, paste your current `requirements` and controls and I’ll tailor the README more precisely.
